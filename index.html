Правила использования

В зависимости от особенностей площадки размещения можно выбрать одну из представленных версий логотипа. Для размещения на темном фоне используется белая версия логотипа, для размещения на белом - синяя версия логотипа. Темная версия логотипа сделана для печати. Минимальный отступ вокруг логотипа - 20px(для печати - 0.3cm)
